# Maze-Adventure-Game
A terminal-based role-playing adventure game developed in Java. Players can create their own characters, choose unique special abilities, explore dangerous mazes, and battle a variety of monsters while progressing through their adventure.

Features

- Character creation system
- Unique special abilities for different character types
- Maze exploration gameplay
- Multiple monster encounters
- Turn-based combat system
- Health and damage mechanics
- Object-oriented design

Technologies

- Java
- Object-Oriented Programming (OOP)
- IntelliJ IDEA
- JUnit

How to Run

1. Clone or download this repository.
2. Open the project in IntelliJ IDEA.
3. Build the project.
4. Run the main game driver class.
5. Follow the instructions displayed in the terminal.

Project Structure

- Character classes
- Monster classes
- Maze and room system
- Combat mechanics
- Game management system
